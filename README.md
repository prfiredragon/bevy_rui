# bevy_rui

These are the 3 demos included. for the game demo you need the [kenney ui assets pack ](https://https://kenney-assets.itch.io/ui-pack)

* the texture for the widgets are not finish.


## Widgets

* Acordion menu
* Button
* Checkbox
* dock
* dropdown
* Label
* Menu
* Slider
* Tabs
* Textbox
* Tooltip
* Viewport
* Windows
